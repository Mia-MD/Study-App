# Study App

Here is a cute Electron app I made to help with studying! There are 3 main features:

## Timer
The user can select how much time they wish to study for with a timer. The user can also enter if they wish to create a pomodoro timer with a customizable amount of sessions.

## Flashcards
The user can create flashcards to help them study for specific subjects.

## To Do
The user can create a to do list with tasks they wish to complish.

## Running the App
If you are directly downloading from the repo, you may use `npm start` to run the app. Besides that, simply launching the app will work.
